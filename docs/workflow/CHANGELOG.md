## 更新原则

1. **及时记录**: 每次更新文档时都要记录
2. **详细说明**: 说明变更的原因和影响
3. **版本关联**: 如果涉及版本号，要更新版本
4. **向后兼容**: 重大变更要考虑向后兼容性

## 如何更新

1. 在相应的文档文件中进行修改
2. 在此文件中添加更新记录
3. 如果涉及核心规则（`.cursorrules`），更新版本号
4. 在相关文档中更新引用（如果有）
