# 透明窗口同步实现方案

## 原理

既然 macOS 不允许跨进程窗口嵌入，我们使用窗口同步方案：

1. **MPV 窗口**：独立原生窗口，性能最好
2. **Electron 窗口**：透明背景，只显示控制面板
3. **窗口同步**：两个窗口位置和大小完全同步

## 实现步骤

### 1. Electron 窗口设为透明
- `transparent: true`
- 控制面板使用不透明背景

### 2. 找到 MPV 窗口
- 通过窗口标题或进程 PID 找到 MPV 窗口

### 3. 同步窗口位置和大小
- 监听 Electron 窗口的 `move` 和 `resize` 事件
- 实时更新 MPV 窗口的位置和大小
